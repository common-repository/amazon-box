== Description ==

Questo plugin altamente personalizzabile permette di inserire in ogni pagina un box contenetnte publicita' dell'affiliazione di Amazon, o altri codici.

== Installation ==

1. Carica il file all'interno di `/wp-content/plugins/`.
2. Nella schermata plugin del lato amministrativo noterai la scritta amazon-box, attiva il plugin.
